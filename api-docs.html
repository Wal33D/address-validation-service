Location Correction Service API

Address validation and geocoding service powered by USPS and Google Maps

Quick Start

The Location Correction Service provides robust address validation and geocoding capabilities with the following features:

  • Address Validation - Validates and standardizes addresses using USPS API
  • Geocoding - Adds geographic coordinates using Google Maps API
  • Smart Preprocessing - Automatically fixes common address format issues
  • Batch Processing - Process up to 100 addresses in a single request
  • High Performance - Request deduplication, caching, and connection pooling
  • Resilient - Circuit breaker pattern prevents cascading failures

Example Request

curl -X POST http://localhost:3715/validate-location \ -H "Content-Type: application/json" \ -d '{ "streetAddress": "1600 Pennsylvania Ave NW", "city": "Washington", "state": "DC", "zipCode": "20500" }'

Example Response

{ "streetAddress": "1600 Pennsylvania Avenue Northwest", "city": "Washington", "state": "DC", "zipCode": "20500", "formattedAddress": "1600 Pennsylvania Ave NW, Washington, DC 20500", "unformattedAddress": "1600 Pennsylvania Ave NW, Washington, DC, 20500", "geo": { "type": "Point", "coordinates": [-77.0365461, 38.89771400000001] }, "status": true }
